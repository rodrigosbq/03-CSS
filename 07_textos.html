TEXTOS NO CSS

É possível importar diversos tipos de letras pelo google e disponibilizar em seu site atraves do https://fonts.google.com/.
Ao escolher uma fonte, é possível também escolher uma família ou estilos , em geral clica-se em view all styles .
Após selecionar os estilos da fonte desejada a esquerda é gerada duas opções de uso da letra:
- A primeira é através de link, para isso deve ser adicionado o código "link" gerado pelo referido site no head do html e em seguida indicar no styles onde será aplicada a letra.
- A segunda forma é atraves de import dispensando a necessidade de incluir no head do projeto , podendo ser colocado diretamente no codigo do CSS.

Estilos utilizados no paragrado acima:
font-size: 26px; /* tamanho*/
font-weight: 500; /* espessura*/
font-family: 'Pacifico', cursive; /* uso do import de estilo de letra */
line-height: 80px; /* espaçamento entre as linhas */
letter-spacing: 8px;/* espaçamento entre as letras */
word-spacing: 30px;/* espaçamento entre as palavras */


Para o alinhamento de texto utilizamos o text-align: que pode ser alinhado de várias formas como center; left; rigth e justify.
Neste caso alinhamos como center

É possível ainda transformar todas as letras em maiusculas, minusculas ou a primeira letra de cada palavra como maiuscula com o text-transform:.
neste caso utilizamos o comando "text-transform: uppercase;".

Função --> text-decoration: underline;

A função text-shadow, seguem 4 parametros sendo a sequencia de horizontal, vertical, esfumaçado e cor.
Segue exemplo: text-shadow: 5px 5px 3px blue;
