Sobre Cores no CSS

As 02 formas mais utilizadas para ajustar cor no HTML são pelo sistema RGB e Hexadecimal.
RGB BASE 10 (0, 1, .... 9) BRANCO (255 255 255) Hexadecimal BASE 16 ( 0, 1, .... 9, A, B , C, D, F) BRANCO (FFFFFF) dica Instalar a extensão Color Highlight.
Sites para "brincar" com cores:
- Para cores basicas: https://www.w3schools.com/colors/colors_rgb.asp
- Para paleta de cores definidas ou a partir de imagens: https://color.adobe.com/pt/create/color-wheel
- Para geração de paletas automatizadas : https://coolors.co/ffffff-81f4e1-56cbf9-ff729f-d3c4d1
- Extensão do Chrome para pegar cores de sites ColorZilla
