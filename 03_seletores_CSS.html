Seletores CSS

Para utilzar um seletor pode ser escolhido por id ou class.
O id é uma identificação única e para ser chamado no CSS utilza-se o nome do id precedido de #.
Ex: no HTML - <h1 id="titulo">Seletores CSS</h1>
no CSS - #titulo{ color: blueviolet; font-size: 55px; }

A class é uma identificação de forma mais generalizada e para ser chamado no CSS utilza-se o nome da class precedido de ".".
Ex: no HTML - <p class="texto">
no CSS - .texto{ color: darkolivegreen; font-style: oblique; font-size: 30px; } }

O seletor universal é o asterisco *
